# Running dev

## Start the backend

```bash

  cd api
  npm install
  npm start

```

### Running backend on debug mode

Debug mode will show debug messages

**Windows**

```bash
  $env:DEBUG='*';npm start
```

**Linux (should verify)**

```bash
  DEBUG='*' npm start
```

see more https://www.npmjs.com/package/debug

## Start the frontend

```bash

  cd client
  npm install
  npm start

```

## Start the backend

```bash
   cd backend
   npm install
   node index.js
```
